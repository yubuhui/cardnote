## 一、华为云同步方案

HarmonyOS 提供了多层次的云同步能力，根据数据形态不同，可选择不同的技术路线：

| 数据形态 | 推荐方案 | 适用场景 |
|---------|---------|---------|
| 结构化数据（笔记、设置等） | 端云数据同步（ArkData + 云空间） | 数据库记录的跨设备同步 |
| 文件/附件（图片、文档等） | 云存储（Cloud Foundation Kit / AGC CloudStorage） | 文件级上传下载 |
| 端云一体化 | AGC Serverless 云开发 | 认证+数据库+存储一站式 |

### 1. 端云数据同步（结构化数据）

利用 ArkData 关系型数据库与华为云空间深度集成，实现数据库记录的自动/手动云端同步。

#### 1.1 云侧配置（AGC 平台）

在 AppGallery Connect 控制台：
1. 进入 **构建 > 云空间服务**，创建容器（容器名需与本地数据库名一致，如 `note`）
2. 配置数据类型，定义字段及类型映射：

| 云侧字段类型 | 本地字段类型 |
|------------|------------|
| String / Encrypted String | TEXT |
| Integer | INTEGER |
| Double | DOUBLE |
| Bytes | BLOB |

3. 部署到生产环境

#### 1.2 端侧配置

**第一步：声明云同步能力**

在 `AppScope/app.json5` 中：

```json
{
  "app": {
    "cloudStructuredDataSyncEnabled": true
  }
}
```

**第二步：创建数据库并设置分布式表**

```typescript
// entry/src/main/ets/database/CloudSyncManager.ets
import { relationalStore } from '@kit.ArkData';
import { BusinessError } from '@kit.BasicServicesKit';

export class CloudSyncManager {
  private store: relationalStore.RdbStore | undefined = undefined;

  // 初始化数据库
  async initDatabase(context: Context): Promise<void> {
    const STORE_CONFIG: relationalStore.StoreConfig = {
      name: 'note.db',           // 数据库名，需与AGC容器名一致
      securityLevel: relationalStore.SecurityLevel.S1
    };

    this.store = await relationalStore.getRdbStore(context, STORE_CONFIG);

    // 创建笔记表
    await this.store.executeSql(
      `CREATE TABLE IF NOT EXISTS Note (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        title TEXT,
        content TEXT,
        createTime INTEGER,
        updateTime INTEGER,
        isDeleted INTEGER DEFAULT 0
      )`
    );

    // 设置分布式表（启用端云同步）
    const tables = ['Note'];
    const config: relationalStore.DistributedConfig = {
      autoSync: true  // true=自动同步, false=仅手动同步
    };

    await this.store.setDistributedTables(tables,
      relationalStore.DistributedType.DISTRIBUTED_CLOUD, config);

    console.info('Cloud sync database initialized.');
  }

  // 手动触发同步
  async manualSync(): Promise<void> {
    if (!this.store) return;

    const tables = ['Note'];
    this.store.cloudSync(
      relationalStore.SyncMode.SYNC_MODE_TIME_FIRST,
      tables,
      (progress: relationalStore.ProgressDetails) => {
        console.info(`Sync progress: ${JSON.stringify(progress)}`);
      },
      (err: BusinessError) => {
        if (err) {
          console.error(`Cloud sync failed: ${err.code}, ${err.message}`);
        } else {
          console.info('Cloud sync completed.');
        }
      }
    );
  }

  // 监听远程数据变更
  subscribeRemoteChange(): void {
    if (!this.store) return;

    this.store.on('dataChange',
      relationalStore.SubscribeType.SUBSCRIBE_TYPE_REMOTE,
      (storeObserver) => {
        console.info('Remote data changed, refreshing local data...');
        // 刷新 UI 数据
      }
    );
  }

  // 插入笔记
  async insertNote(title: string, content: string): Promise<void> {
    if (!this.store) return;
    const now = Date.now();
    await this.store.executeSql(
      'INSERT INTO Note (title, content, createTime, updateTime) VALUES (?, ?, ?, ?)',
      [title, content, now, now]
    );
  }

  // 查询所有笔记
  async queryAllNotes(): Promise<Array<Record<string, unknown>>> {
    if (!this.store) return [];
    const predicates = new relationalStore.RdbPredicates('Note');
    predicates.equalTo('isDeleted', 0);
    const resultSet = await this.store.query(predicates);

    const results: Array<Record<string, unknown>> = [];
    while (resultSet.goToNextRow()) {
      results.push({
        id: resultSet.getLong(resultSet.getColumnIndex('id')),
        title: resultSet.getString(resultSet.getColumnIndex('title')),
        content: resultSet.getString(resultSet.getColumnIndex('content')),
        updateTime: resultSet.getLong(resultSet.getColumnIndex('updateTime')),
      });
    }
    resultSet.close();
    return results;
  }
}
```

### 2. 云存储（文件同步）

使用 Cloud Foundation Kit 的 `cloudStorage` 模块（API 12+）进行文件的上传下载。

```typescript
// entry/src/main/ets/cloud/CloudFileManager.ets
import { cloudStorage } from '@kit.CloudFoundationKit';
import { BusinessError, request } from '@kit.BasicServicesKit';
import { common } from '@kit.AbilityKit';

export class CloudFileManager {
  private bucket: cloudStorage.StorageBucket;
  private context: common.UIAbilityContext;

  constructor(context: common.UIAbilityContext) {
    this.context = context;
    this.bucket = cloudStorage.bucket(); // 使用默认存储实例
  }

  // 上传文件
  async uploadFile(localFileName: string, cloudDir: string = ''): Promise<void> {
    const localPath = `${this.context.cacheDir}/${localFileName}`;
    // 云侧路径：支持 "目录/文件名" 格式
    const cloudPath = cloudDir ? `${cloudDir}/${localFileName}` : localFileName;

    return new Promise<void>((resolve, reject) => {
      this.bucket.uploadFile(this.context, {
        localPath: localPath,
        cloudPath: cloudPath
      }).then((task: request.agent.Task) => {

        task.on('progress', (progress) => {
          console.info(`Upload progress: ${JSON.stringify(progress)}`);
        });

        task.on('completed', (progress) => {
          console.info('Upload completed.');
          resolve();
        });

        task.on('failed', (progress) => {
          console.error('Upload failed:', JSON.stringify(progress));
          reject(new Error('Upload failed'));
        });

        task.start((err: BusinessError) => {
          if (err) {
            console.error(`Failed to start upload: ${err.message}`);
            reject(err);
          }
        });

      }).catch((err: BusinessError) => {
        console.error(`Upload error: ${err.message}`);
        reject(err);
      });
    });
  }

  // 下载文件
  async downloadFile(cloudPath: string, localFileName?: string): Promise<string> {
    const fileName = localFileName || cloudPath.split('/').pop() || 'download';
    const localPath = `${this.context.cacheDir}/${fileName}`;

    return new Promise<string>((resolve, reject) => {
      this.bucket.downloadFile(this.context, {
        cloudPath: cloudPath,
        localPath: localPath
      }).then((task: request.agent.Task) => {

        task.on('completed', () => {
          console.info('Download completed.');
          resolve(localPath);
        });

        task.on('failed', (progress) => {
          reject(new Error('Download failed'));
        });

        task.start((err: BusinessError) => {
          if (err) reject(err);
        });

      }).catch((err: BusinessError) => {
        reject(err);
      });
    });
  }

  // 获取文件列表
  async listFiles(cloudPath: string = ''): Promise<cloudStorage.ListResults> {
    try {
      const result = await this.bucket.list(cloudPath);
      console.info(`File list: ${JSON.stringify(result)}`);
      return result;
    } catch (err) {
      console.error(`List files error: ${JSON.stringify(err)}`);
      throw err;
    }
  }

  // 获取文件下载地址
  async getDownloadURL(cloudPath: string): Promise<string> {
    try {
      const url = await this.bucket.getDownloadURL(cloudPath);
      return url;
    } catch (err) {
      console.error(`Get download URL error: ${JSON.stringify(err)}`);
      throw err;
    }
  }

  // 删除云侧文件
  async deleteFile(cloudPath: string): Promise<void> {
    try {
      await this.bucket.deleteFile(cloudPath);
      console.info('File deleted successfully.');
    } catch (err) {
      console.error(`Delete file error: ${JSON.stringify(err)}`);
      throw err;
    }
  }

  // 获取文件元数据
  async getMetadata(cloudPath: string): Promise<cloudStorage.Metadata> {
    try {
      const metadata = await this.bucket.getMetadata(cloudPath);
      return metadata;
    } catch (err) {
      console.error(`Get metadata error: ${JSON.stringify(err)}`);
      throw err;
    }
  }
}
```

## 二、WebDAV 同步方案

WebDAV 本质上是 HTTP 协议的扩展，可使用 `@ohos.net.http` 模块自行实现 WebDAV 客户端。也可使用社区提供的基础库，但核心逻辑均基于 HTTP XML 请求。

### WebDAV 协议核心方法一览

| 操作 | HTTP 方法 | 说明 |
|------|----------|------|
| 列出目录 | `PROPFIND` | 获取目录下的文件/文件夹列表 |
| 下载文件 | `GET` | 下载指定文件 |
| 上传文件 | `PUT` | 上传文件到指定路径 |
| 创建目录 | `MKCOL` | 创建新文件夹 |
| 删除文件 | `DELETE` | 删除指定文件/文件夹 |
| 移动/重命名 | `MOVE` | 移动或重命名文件 |

### WebDAV 客户端封装

```typescript
// entry/src/main/ets/webdav/WebDAVClient.ets
import { http } from '@kit.NetworkKit';
import { BusinessError } from '@kit.BasicServicesKit';
import { util } from '@kit.ArkTS';

/**
 * WebDAV 文件/文件夹信息
 */
export interface WebDAVItem {
  name: string;           // 文件/文件夹名称
  path: string;           // 完整路径
  isDirectory: boolean;   // 是否为目录
  size: number;           // 文件大小（字节）
  lastModified: string;   // 最后修改时间
  contentType: string;    // 内容类型
}

export class WebDAVClient {
  private baseUrl: string;
  private username: string;
  private password: string;
  private authHeader: string;

  /**
   * 构造函数
   * @param baseUrl WebDAV 服务器地址，如 https://dav.example.com/remote.php/dav/files/user/
   * @param username 用户名
   * @param password 密码
   */
  constructor(baseUrl: string, username: string, password: string) {
    // 确保 URL 以 '/' 结尾
    this.baseUrl = baseUrl.endsWith('/') ? baseUrl : baseUrl + '/';
    this.username = username;
    this.password = password;

    // 生成 Basic Auth 头部
    const credentials = `${username}:${password}`;
    const base64 = new util.Base64Helper();
    const encoded = base64.encodeToStringSync(
      new Uint8Array(Buffer.from(credentials).buffer)
    );
    this.authHeader = `Basic ${encoded}`;
  }

  /**
   * 发送 WebDAV 请求的通用方法
   */
  private async sendRequest(
    method: http.RequestMethod,
    path: string,
    options?: {
      body?: string | ArrayBuffer;
      depth?: string;
      extraHeaders?: Record<string, string>;
    }
  ): Promise<http.HttpResponse> {
    const url = this.getFullUrl(path);

    const headers: Record<string, string> = {
      'Authorization': this.authHeader,
      'User-Agent': 'HarmonyOS-WebDAV-Client/1.0',
    };

    if (options?.depth) {
      headers['Depth'] = options.depth;
    }
    if (options?.body && typeof options.body === 'string') {
      headers['Content-Type'] = 'application/xml; charset=utf-8';
    }
    if (options?.extraHeaders) {
      Object.keys(options.extraHeaders).forEach(key => {
        headers[key] = options.extraHeaders![key];
      });
    }

    const requestOptions: http.HttpRequestOptions = {
      method: method,
      header: headers,
      connectTimeout: 30000,
      readTimeout: 60000,
    };

    if (options?.body) {
      requestOptions.extraData = options.body;
    }

    const httpRequest = http.createHttp();

    try {
      const response = await httpRequest.request(url, requestOptions);
      return response;
    } finally {
      httpRequest.destroy();
    }
  }

  private getFullUrl(path: string): string {
    if (path.startsWith('/')) {
      path = path.substring(1);
    }
    return this.baseUrl + path;
  }

  /**
   * 列出目录内容
   */
  async listDirectory(dirPath: string = ''): Promise<WebDAVItem[]> {
    const propfindBody = `<?xml version="1.0" encoding="utf-8"?>
      <d:propfind xmlns:d="DAV:">
        <d:prop>
          <d:displayname/>
          <d:getcontentlength/>
          <d:getlastmodified/>
          <d:getcontenttype/>
          <d:resourcetype/>
        </d:prop>
      </d:propfind>`;

    const response = await this.sendRequest(
      http.RequestMethod.PROPFIND,
      dirPath,
      { body: propfindBody, depth: '1' }
    );

    return this.parsePropfindResponse(response.result as string);
  }

  /**
   * 解析 PROPFIND 响应
   */
  private parsePropfindResponse(xmlString: string): WebDAVItem[] {
    const items: WebDAVItem[] = [];

    // 使用正则提取每个 response 元素
    const responseRegex = /<d:response[^>]*>([\s\S]*?)<\/d:response>/g;
    let match: RegExpExecArray | null;

    while ((match = responseRegex.exec(xmlString)) !== null) {
      const responseBlock = match[1];

      // 提取路径
      const hrefMatch = /<d:href>(.+?)<\/d:href>/.exec(responseBlock);
      const href = hrefMatch ? this.decodeXml(hrefMatch[1]) : '';

      // 提取显示名称
      const nameMatch = /<d:displayname>(.*?)<\/d:displayname>/.exec(responseBlock);
      const name = nameMatch ? this.decodeXml(nameMatch[1]) : '';

      // 判断是否为目录
      const isDirectory = /<d:collection\s*\/>/.test(responseBlock) ||
        /<d:collection><\/d:collection>/.test(responseBlock);

      // 提取文件大小
      const sizeMatch = /<d:getcontentlength>(\d+)<\/d:getcontentlength>/.exec(responseBlock);
      const size = sizeMatch ? parseInt(sizeMatch[1]) : 0;

      // 提取最后修改时间
      const timeMatch = /<d:getlastmodified>(.+?)<\/d:getlastmodified>/.exec(responseBlock);
      const lastModified = timeMatch ? this.decodeXml(timeMatch[1]) : '';

      // 提取内容类型
      const typeMatch = /<d:getcontenttype>(.*?)<\/d:getcontenttype>/.exec(responseBlock);
      const contentType = typeMatch ? this.decodeXml(typeMatch[1]) : '';

      if (name && name !== this.baseUrl.split('/').filter(s => s).pop()) {
        items.push({
          name: name,
          path: href,
          isDirectory: isDirectory,
          size: size,
          lastModified: lastModified,
          contentType: contentType,
        });
      }
    }

    return items;
  }

  /**
   * XML 实体解码
   */
  private decodeXml(str: string): string {
    return str
      .replace(/&amp;/g, '&')
      .replace(/&lt;/g, '<')
      .replace(/&gt;/g, '>')
      .replace(/&quot;/g, '"')
      .replace(/&apos;/g, "'");
  }

  /**
   * 下载文件到本地
   */
  async downloadFile(
    remotePath: string,
    localPath: string,
    onProgress?: (loaded: number, total: number) => void
  ): Promise<void> {
    return new Promise<void>(async (resolve, reject) => {
      try {
        const url = this.getFullUrl(remotePath);
        const httpRequest = http.createHttp();

        httpRequest.on('dataReceiveProgress', (data: http.DataReceiveProgressInfo) => {
          onProgress?.(data.receiveSize, data.totalSize);
        });

        const response = await httpRequest.request(url, {
          method: http.RequestMethod.GET,
          header: {
            'Authorization': this.authHeader,
            'User-Agent': 'HarmonyOS-WebDAV-Client/1.0',
          },
          readTimeout: 120000,
        });

        if (response.responseCode === 200 && response.result instanceof ArrayBuffer) {
          // 写入本地文件
          const { fileIo as fs } = await import('@kit.CoreFileKit');
          const file = fs.openSync(localPath, fs.OpenMode.CREATE | fs.OpenMode.WRITE_ONLY);
          fs.writeSync(file.fd, response.result as ArrayBuffer);
          fs.closeSync(file.fd);
          resolve();
        } else {
          reject(new Error(`Download failed. HTTP ${response.responseCode}`));
        }

        httpRequest.destroy();
      } catch (err) {
        reject(err);
      }
    });
  }

  /**
   * 上传文件（读取本地文件内容并 PUT 到服务器）
   */
  async uploadFile(
    localPath: string,
    remotePath: string,
    contentType?: string,
    onProgress?: (loaded: number, total: number) => void
  ): Promise<void> {
    const { fileIo as fs } = await import('@kit.CoreFileKit');

    // 读取本地文件
    const file = fs.openSync(localPath, fs.OpenMode.READ_ONLY);
    const stat = fs.statSync(file.fd);
    const buffer = new ArrayBuffer(stat.size);
    fs.readSync(file.fd, buffer);
    fs.closeSync(file.fd);

    const response = await this.sendRequest(
      http.RequestMethod.PUT,
      remotePath,
      {
        body: buffer,
        extraHeaders: {
          'Content-Type': contentType || 'application/octet-stream',
          'Content-Length': stat.size.toString(),
        }
      }
    );

    if (response.responseCode !== 200 && response.responseCode !== 201 &&
      response.responseCode !== 204) {
      throw new Error(`Upload failed. HTTP ${response.responseCode}`);
    }
  }

  /**
   * 直接上传字符串/文本内容
   */
  async uploadText(remotePath: string, content: string, contentType: string = 'text/plain'): Promise<void> {
    const response = await this.sendRequest(
      http.RequestMethod.PUT,
      remotePath,
      {
        body: content,
        extraHeaders: {
          'Content-Type': contentType,
        }
      }
    );

    if (response.responseCode !== 200 && response.responseCode !== 201 &&
      response.responseCode !== 204) {
      throw new Error(`Upload text failed. HTTP ${response.responseCode}`);
    }
  }

  /**
   * 创建目录
   */
  async createDirectory(dirPath: string): Promise<void> {
    const response = await this.sendRequest(
      http.RequestMethod.MKCOL,
      dirPath
    );

    if (response.responseCode !== 201) {
      throw new Error(`Create directory failed. HTTP ${response.responseCode}`);
    }
  }

  /**
   * 删除文件或目录
   */
  async delete(path: string): Promise<void> {
    const response = await this.sendRequest(
      http.RequestMethod.DELETE,
      path
    );

    if (response.responseCode !== 200 && response.responseCode !== 204) {
      throw new Error(`Delete failed. HTTP ${response.responseCode}`);
    }
  }

  /**
   * 移动或重命名文件
   */
  async move(sourcePath: string, destPath: string): Promise<void> {
    const destUrl = this.getFullUrl(destPath);
    const response = await this.sendRequest(
      http.RequestMethod.MOVE,
      sourcePath,
      {
        extraHeaders: {
          'Destination': destUrl,
        }
      }
    );

    if (response.responseCode !== 201 && response.responseCode !== 204) {
      throw new Error(`Move failed. HTTP ${response.responseCode}`);
    }
  }

  /**
   * 检查文件是否存在
   */
  async exists(remotePath: string): Promise<boolean> {
    const response = await this.sendRequest(
      http.RequestMethod.HEAD,
      remotePath
    );
    return response.responseCode === 200;
  }
}
```

### WebDAV 使用示例

```typescript
// entry/src/main/ets/pages/WebDAVSyncPage.ets
import { WebDAVClient, WebDAVItem } from '../webdav/WebDAVClient';
import { common } from '@kit.AbilityKit';

@Entry
@Component
struct WebDAVSyncPage {
  @State items: WebDAVItem[] = [];
  @State currentPath: string = '';
  @State isLoading: boolean = false;
  @State statusText: string = '就绪';

  private client: WebDAVClient = new WebDAVClient(
    'https://your-webdav-server.com/remote.php/dav/files/username/',
    'your-username',
    'your-password'
  );

  aboutToAppear(): void {
    this.loadDirectory('');
  }

  // 加载目录
  async loadDirectory(path: string): Promise<void> {
    this.isLoading = true;
    this.statusText = '加载中...';
    try {
      this.items = await this.client.listDirectory(path);
      this.currentPath = path;
      this.statusText = `共 ${this.items.length} 个项目`;
    } catch (err) {
      this.statusText = `加载失败: ${JSON.stringify(err)}`;
    } finally {
      this.isLoading = false;
    }
  }

  // 上传本地文件
  async handleUpload(): Promise<void> {
    try {
      const context = this.getUIContext().getHostContext() as common.UIAbilityContext;
      const localPath = `${context.filesDir}/note_backup.json`;
      const remotePath = this.currentPath
        ? `${this.currentPath}/note_backup.json`
        : 'note_backup.json';

      this.statusText = '上传中...';
      await this.client.uploadFile(localPath, remotePath, 'application/json');
      this.statusText = '上传成功';

      // 刷新目录
      await this.loadDirectory(this.currentPath);
    } catch (err) {
      this.statusText = `上传失败: ${JSON.stringify(err)}`;
    }
  }

  // 下载文件
  async handleDownload(item: WebDAVItem): Promise<void> {
    if (item.isDirectory) {
      await this.loadDirectory(item.path);
      return;
    }
    try {
      const context = this.getUIContext().getHostContext() as common.UIAbilityContext;
      const localPath = `${context.cacheDir}/${item.name}`;

      this.statusText = `下载中: ${item.name}...`;
      await this.client.downloadFile(item.path, localPath, (loaded, total) => {
        const progress = ((loaded / total) * 100).toFixed(1);
        this.statusText = `下载进度: ${progress}%`;
      });
      this.statusText = `下载完成: ${item.name}`;
    } catch (err) {
      this.statusText = `下载失败: ${JSON.stringify(err)}`;
    }
  }

  build() {
    Column() {
      // 状态栏
      Row() {
        Text(this.statusText)
          .fontSize(14)
          .fontColor('#666')
          .layoutWeight(1)
        Button('上传')
          .onClick(() => this.handleUpload())
          .fontSize(14)
```

## 三、统一同步架构设计

将华为云同步和 WebDAV 同步整合为统一的同步管理器：

```typescript
// entry/src/main/ets/sync/SyncManager.ets
import { CloudSyncManager } from '../database/CloudSyncManager';
import { CloudFileManager } from '../cloud/CloudFileManager';
import { WebDAVClient } from '../webdav/WebDAVClient';

export enum SyncProvider {
  HUAWEI_CLOUD = 'huawei_cloud',
  WEBDAV = 'webdav',
}

export class SyncManager {
  private cloudSync: CloudSyncManager;
  private cloudFile: CloudFileManager;
  private webdavClient: WebDAVClient | null = null;

  constructor(context: Context) {
    this.cloudSync = new CloudSyncManager();
    this.cloudFile = new CloudFileManager(context);
  }

  // 配置 WebDAV
  configureWebDAV(url: string, username: string, password: string): void {
    this.webdavClient = new WebDAVClient(url, username, password);
  }

  // 执行同步
  async sync(provider: SyncProvider): Promise<void> {
    switch (provider) {
      case SyncProvider.HUAWEI_CLOUD:
        await this.cloudSync.manualSync();
        break;
      case SyncProvider.WEBDAV:
        if (!this.webdavClient) {
          throw new Error('WebDAV not configured.');
        }
        await this.syncToWebDAV();
        break;
    }
  }

  // 同步数据到 WebDAV
  private async syncToWebDAV(): Promise<void> {
    if (!this.webdavClient) return;

    // 1. 导出数据库数据为 JSON
    const notes = await this.cloudSync.queryAllNotes();
    const jsonContent = JSON.stringify(notes, null, 2);

    // 2. 上传到 WebDAV
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
    await this.webdavClient.uploadText(
      `backup/notes_${timestamp}.json`,
      jsonContent,
      'application/json'
    );
  }
}
```

| 维度 | 华为云同步 <rsup>1</rsup><rsup>2</rsup>| WebDAV 同步 |
|------|----------|------------|
| **结构化数据** | ArkData + 云空间（自动/手动同步） | 导出 JSON 后 PUT 上传（需自行管理版本） |
| **文件同步** | Cloud Foundation Kit (uploadFile/downloadFile) | HTTP PUT/GET 方法 |
| **认证** | 华为账号自动关联 | Basic Auth / Digest Auth |
| **多设备** | 端-云-端自动协同 | 各设备独立连接 WebDAV 服务器 |
| **开箱即用** | AGC 配置后直接使用 | 需自行搭建或购买 WebDAV 服务 |
| **灵活性** | 深度绑定华为生态 | 可对接任意 WebDAV 服务商（Nextcloud、ownCloud 等） |

```typescript
import { relationalStore } from '@kit.ArkData';
import { BusinessError } from '@kit.BasicServicesKit';
import { Note } from '../model/NoteModel';

const DB_NAME = 'note.db';      // ← 这个名称要跟 AGC 云空间容器的名称一致
const TABLE_NAME = 'Note';

export class NoteDatabase {
  private store: relationalStore.RdbStore | undefined;

  /** 初始化数据库（在 EntryAbility.onCreate 中调用一次） */
  async init(context: Context): Promise<void> {
    const config: relationalStore.StoreConfig = {
      name: DB_NAME,
      securityLevel: relationalStore.SecurityLevel.S1,
    };

    this.store = await relationalStore.getRdbStore(context, config);

    // 建表（所有字段不能设 NOT NULL，云同步要求）
    await this.store.executeSql(`
      CREATE TABLE IF NOT EXISTS ${TABLE_NAME} (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        title TEXT,
        content TEXT,
        createTime INTEGER,
        updateTime INTEGER,
        isDeleted INTEGER DEFAULT 0
      )
    `);

    // 设置分布式表 → 启用华为云同步
    await this.store.setDistributedTables(
      [TABLE_NAME],
      relationalStore.DistributedType.DISTRIBUTED_CLOUD,
      { autoSync: true }   // true=自动同步, false=仅手动
    );

    console.info('NoteDatabase initialized with cloud sync.');
  }

  /** 获取 store 实例（供 Repository 使用） */
  getStore(): relationalStore.RdbStore | undefined {
    return this.store;
  }
}
```

### 4.3 `database/NoteRepository.ets` — 数据增删改查

这是你**现有代码中所有操作数据库的地方要改指向的"新入口"**：

```typescript
// entry/src/main/ets/database/NoteRepository.ets
import { relationalStore } from '@kit.ArkData';
import { noteDatabase } from './NoteDatabase';
import { Note } from '../model/NoteModel';

export class NoteRepository {

  /** 查询所有未删除的笔记 */
  async queryAll(): Promise<Note[]> {
    const store = noteDatabase.getStore();
    if (!store) return [];

    const predicates = new relationalStore.RdbPredicates('Note');
    predicates.equalTo('isDeleted', 0);
    predicates.orderByDesc('updateTime');

    const resultSet = await store.query(predicates);
    const notes: Note[] = [];
    while (resultSet.goToNextRow()) {
      notes.push(this.rowToNote(resultSet));
    }
    resultSet.close();
    return notes;
  }

  /** 插入笔记，返回自增 id */
  async insert(note: Note): Promise<number> {
    const store = noteDatabase.getStore();
    if (!store) return -1;

    const now = Date.now();
    const row = await store.insert('Note', {
      title: note.title,
      content: note.content,
      createTime: now,
      updateTime: now,
      isDeleted: 0,
    });
    return row;
  }

  /** 更新笔记 */
  async update(note: Note): Promise<void> {
    const store = noteDatabase.getStore();
    if (!store) return;

    const values: Record<string, ValueType> = {
      title: note.title,
      content: note.content,
      updateTime: Date.now(),
    };

    const predicates = new relationalStore.RdbPredicates('Note');
    predicates.equalTo('id', note.id);
    await store.update(values, predicates);
  }

  /** 软删除 */
  async softDelete(id: number): Promise<void> {
    const store = noteDatabase.getStore();
    if (!store) return;

    const values: Record<string, ValueType> = {
      isDeleted: 1,
      updateTime: Date.now(),
    };
    const predicates = new relationalStore.RdbPredicates('Note');
    predicates.equalTo('id', id);
    await store.update(values, predicates);
  }

  /** 手动触发云同步 */
  async manualCloudSync(): Promise<void> {
    const store = noteDatabase.getStore();
    if (!store) return;

    return new Promise<void>((resolve, reject) => {
      store.cloudSync(
        relationalStore.SyncMode.SYNC_MODE_TIME_FIRST,
        ['Note'],
        (progress) => console.info('Sync progress:', JSON.stringify(progress)),
        (err) => {
          if (err) {
            console.error('Cloud sync failed:', err.message);
            reject(err);
          } else {
            console.info('Cloud sync completed.');
            resolve();
          }
        }
      );
    });
  }

  /** 导出所有笔记为 JSON 字符串（供 WebDAV 上传） */
  async exportToJson(): Promise<string> {
    const notes = await this.queryAll();
    return JSON.stringify(notes, null, 2);
  }

  /** 从 JSON 字符串导入笔记（从 WebDAV 下载后恢复） */
  async importFromJson(json: string): Promise<void> {
    const notes: Note[] = JSON.parse(json);
    for (const note of notes) {
      await this.insert(note);
    }
  }

  private rowToNote(rs: relationalStore.ResultSet): Note {
    return {
      id: rs.getLong(rs.getColumnIndex('id')),
      title: rs.getString(rs.getColumnIndex('title')),
      content: rs.getString(rs.getColumnIndex('content')),
      createTime: rs.getLong(rs.getColumnIndex('createTime')),
      updateTime: rs.getLong(rs.getColumnIndex('updateTime')),
      isDeleted: rs.getLong(rs.getColumnIndex('isDeleted')),
    };
  }
}

export const noteRepo = new NoteRepository();
```

### 4.4 `webdav/WebDAVClient.ets` — WebDAV 客户端

```typescript
// entry/src/main/ets/webdav/WebDAVClient.ets
import { http } from '@kit.NetworkKit';
import { util } from '@kit.ArkTS';

export class WebDAVClient {
  private baseUrl: string;
  private authHeader: string;

  constructor(baseUrl: string, username: string, password: string) {
    // 确保以 '/' 结尾
    this.baseUrl = baseUrl.endsWith('/') ? baseUrl : baseUrl + '/';

    // 生成 Basic Auth 头部
    const credentials = `${username}:${password}`;
    const base64 = new util.Base64Helper();
    this.authHeader = 'Basic ' + base64.encodeToStringSync(
      new Uint8Array(Buffer.from(credentials).buffer)
    );
  }

  /** 上传文本内容（如笔记 JSON 备份） */
  async uploadText(remotePath: string, content: string, contentType: string = 'application/json'): Promise<void> {
    const httpRequest = http.createHttp();
    try {
      const response = await httpRequest.request(
        this.baseUrl + remotePath,
        {
          method: http.RequestMethod.PUT,
          header: {
            'Authorization': this.authHeader,
            'Content-Type': contentType,
          },
          extraData: content,
          connectTimeout: 30000,
          readTimeout: 60000,
        }
      );
      if (response.responseCode !== 200 && response.responseCode !== 201 && response.responseCode !== 204) {
        throw new Error(`Upload failed. HTTP ${response.responseCode}`);
      }
    } finally {
      httpRequest.destroy();
    }
  }

  /** 下载文本内容 */
  async downloadText(remotePath: string): Promise<string> {
    const httpRequest = http.createHttp();
    try {
      const response = await httpRequest.request(
        this.baseUrl + remotePath,
        {
          method: http.RequestMethod.GET,
          header: {
            'Authorization': this.authHeader,
          },
          expectDataType: http.HttpDataType.STRING,
          connectTimeout: 30000,
          readTimeout: 60000,
        }
      );
      if (response.responseCode === 200) {
        return response.result as string;
      }
      throw new Error(`Download failed. HTTP ${response.responseCode}`);
    } finally {
      httpRequest.destroy();
    }
  }

  /** 列出目录 */
  async listDirectory(remotePath: string = ''): Promise<{ name: string; isDirectory: boolean; size: number }[]> {
    const body = `<?xml version="1.0" encoding="utf-8"?>
      <d:propfind xmlns:d="DAV:">
        <d:prop>
          <d:displayname/>
          <d:getcontentlength/>
          <d:resourcetype/>
        </d:prop>
      </d:propfind>`;

    const httpRequest = http.createHttp();
    try {
      const response = await httpRequest.request(
        this.baseUrl + remotePath,
        {
          method: http.RequestMethod.PROPFIND,
          header: {
            'Authorization': this.authHeader,
            'Content-Type': 'application/xml; charset=utf-8',
            'Depth': '1',
          },
          extraData: body,
          connectTimeout: 30000,
          readTimeout: 60000,
        }
      );

      if (response.responseCode === 207) {
        return this.parseFileList(response.result as string);
      }
      return [];
    } finally {
      httpRequest.destroy();
    }
  }

  /** 检查连接是否正常 */
  async testConnection(): Promise<boolean> {
    try {
      const httpRequest = http.createHttp();
      try {
        const response = await httpRequest.request(
          this.baseUrl,
          {
            method: http.RequestMethod.OPTIONS,
            header: { 'Authorization': this.authHeader },
            connectTimeout: 10000,
            readTimeout: 10000,
          }
        );
        return response.responseCode >= 200 && response.responseCode < 400;
      } finally {
        httpRequest.destroy();
      }
    } catch {
      return false;
    }
  }

  /* 简易 XML 解析（只提取 displayname、collection、getcontentlength） */
  private parseFileList(xml: string): { name: string; isDirectory: boolean; size: number }[] {
    const items: { name: string; isDirectory: boolean; size: number }[] = [];
    const responseRegex = /<d:response[^>]*>([\s\S]*?)<\/d:response>/g;
    let match: RegExpExecArray | null;

    while ((match = responseRegex.exec(xml)) !== null) {
      const block = match[1];
      const nameMatch = /<d:displayname>(.*?)<\/d:displayname>/.exec(block);
      const isDir = /<d:collection\s*\/>/.test(block) || /<d:collection>/.test(block);
      const sizeMatch = /<d:getcontentlength>(\d+)<\/d:getcontentlength>/.exec(block);

      const name = nameMatch ? this.unescapeXml(nameMatch[1]) : '';
      if (name) {
        items.push({
          name,
          isDirectory: isDir,
          size: sizeMatch ? parseInt(sizeMatch[1]) : 0,
        });
      }
    }
    return items;
  }

  private unescapeXml(str: string): string {
    return str.replace(/&amp;/g, '&').replace(/&lt;/g, '<')
      .replace(/&gt;/g, '>').replace(/&quot;/g, '"').replace(/&apos;/g, "'");
  }
}
```

```typescript
// entry/src/main/ets/sync/SyncManager.ets
import { noteRepo } from '../database/NoteRepository';
import { WebDAVClient } from '../webdav/WebDAVClient';
import { preferences } from '@kit.ArkData';

const PREF_NAME = 'sync_settings';

export enum SyncProvider {
  HUAWEI_CLOUD = 'huawei_cloud',
  WEBDAV = 'webdav',
}

export class SyncManager {
  private webdavClient: WebDAVClient | null = null;

  /** 配置 WebDAV 连接 */
  configureWebDAV(url: string, username: string, password: string): void {
    this.webdavClient = new WebDAVClient(url, username, password);
  }

  /** 测试 WebDAV 连接 */
  async testWebDAV(): Promise<boolean> {
    return this.webdavClient?.testConnection() ?? false;
  }

  /** 执行华为云同步 */
  async syncToCloud(): Promise<void> {
    await noteRepo.manualCloudSync();
  }

  /** 执行 WebDAV 同步（备份 + 恢复） */
  async syncToWebDAV(): Promise<void> {
    if (!this.webdavClient) throw new Error('WebDAV not configured.');

    // 导出本地数据为 JSON，上传到 WebDAV
    const json = await noteRepo.exportToJson();
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
    await this.webdavClient.uploadText(
      `cardnote_backup_${timestamp}.json`,
      json,
      'application/json'
    );
  }

  /** 从 WebDAV 恢复笔记 */
  async restoreFromWebDAV(remotePath: string): Promise<void> {
    if (!this.webdavClient) throw new Error('WebDAV not configured.');
    const json = await this.webdavClient.downloadText(remotePath);
    await noteRepo.importFromJson(json);
  }

  /** 列出 WebDAV 备份文件 */
  async listBackups(): Promise<{ name: string; isDirectory: boolean; size: number }[]> {
    if (!this.webdavClient) return [];
    return this.webdavClient.listDirectory('');
  }

  /** 保存用户配置到本地 */
  async saveConfig(config: { webdavUrl: string; webdavUser: string; webdavPass: string }): Promise<void> {
    const prefs = await preferences.getPreferences(getContext(), PREF_NAME);
    await prefs.put('webdav_url', config.webdavUrl);
    await prefs.put('webdav_user', config.webdavUser);
    await prefs.put('webdav_pass', config.webdavPass);
    await prefs.flush();
  }

  /** 加载本地配置并初始化 WebDAV */
  async loadAndInitWebDAV(): Promise<boolean> {
    const prefs = await preferences.getPreferences(getContext(), PREF_NAME);
    const url = await prefs.get('webdav_url', '') as string;
    const user = await prefs.get('webdav_user', '') as string;
    const pass = await prefs.get('webdav_pass', '') as string;
    if (url && user && pass) {
      this.configureWebDAV(url, user, pass);
      return true;
    }
    return false;
  }
}
```

### 5.2 现有笔记页面 — 替换数据访问

在你的笔记列表/编辑页面中，把原来直接操作数据库的代码，换成通过 `NoteRepository`：

**原代码（假设）：**
```typescript
// 原来可能直接调 store.insert / store.query
this.store.executeSql('INSERT INTO Note ...');
```

**改成：**
```typescript
import { noteRepo } from '../database/NoteRepository';
import { syncManager } from '../sync/SyncManager';

// 加载笔记列表
this.notes = await noteRepo.queryAll();

// 新增笔记
await noteRepo.insert(newNote);

// 更新笔记
await noteRepo.update(editedNote);

// 删除笔记
await noteRepo.softDelete(noteId);
```

> 搜索你项目中 `executeSql` / `relationalStore` 等关键字，把它们逐个替换成 `noteRepo.xxx()`。
